# Collection of Guides

**Collection of Guides** - Коллекция гайдов, советов, помощи  по программирование, безопасности и тд.

## Специализация Коллекции по языкам программированиям

- `Python`: на данный момент будут гайды по Python, в будущем добавлю другие языки программирования (Batch, JavaScript, C++)

- Базовые запросы `SQL`: базовые запросы на SQLite (Python)

- Модуль `cryptography` (Python): RSA, Fernet, Хэширование

- В скором будут много специализации, `но это начало`
