## Начинаем изучение

### Обработка исключений используя `try/except/finally`

- Что такое `try/except`?

- try/except - защита от падения программы, они обрабатывают исключения

- Например, пользователь нажал написал 10 / 0, произойдет исключений ZeroDivisonError - ведь нельзя делить на ноль, но try/except нам помогут

- Посмотрите и сравните:

```python
print(10/0) # Упадёт программа, ошибка: ZeroDivisionError


# Теперь исправленный вариант

try:
    print(10/0)
except ZeroDivisionError:
    print("[-] Нельзя делить на ноль") # Программа продолжит дальше работать
    print("[+] Результат: 0")
```

- Это удобно, ведь используя `try/except` программа устаит перед ошибками

- Если не было обработка исключений, программы падали

- Ведь представьте, если не было обработки исключений (вообще) - то из-за маленькой ошибки ваша Операционная Система упадет - это плохо

- Но, что такое `finally`?

- Его используют вместе с `try` или `try/except`, он нужен для обязательного выполения действий, например, закрыть файл (узнаете позже)

- Пример (узнаете позже как работать):

```python
f = None
try:
    f = open("README.md", 'r'): # Для примера без with 
       contents = f.read()
       print(contents)
    
except FileNotFoundError:
    print("[-] Файл не найден")

finally:
    if f:
        f.close()
```

### Исключения

- Наверное вам надо знать виды исключений

- Но, дам вам фишку, если не хотитие обрабатывать кажое исключений подряд, используйте так:

```python

try:
    pass 
except Exception as e: # Exception - обработает все ошибки и переменная e (используя as e) - покажет какая ошибка
    print(f'[-] Произошла ошибка: {e}')

```

- Ладно, переходим к видам исключений

#### Известный исключения

- SyntaxError: Синтаксическая ошибка

- IndentationError: Ошибка отступа

- NameError: Переменная не определена

- TypeError: Неправильный тип данных

- ValueError: Неправильное значение

- ZeroDivisionError: Нельзя делить на ноль

- FileNotFoundError: Файл не найден

- KeyError: Ключ не найден в словаре

- IndexError: Индекс вне диапозона

- AttributeError: Нет атрибута/метода

- ImportError / ModuleNotFoundError: Модуль не найден

- KeybordInterrupt: Пользователь приостановил программу (Ctrl + C)
