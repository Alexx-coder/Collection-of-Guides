#  Lesson module `sys` of Pythhon

**В данном уроке мы изучим библиотеку sys**

## Начинаем изучение
 
- Что за модуль `sys`? sys - модуль стандартной библиотеки Python, для вазимодействия с интерпортёром Python и системой

- Подключаем как:

```python
import sys
```

### Функции 

- Давайте изучим функции sys:

```python
import sys

sys.argv # Аргументы командной строки

sys.exit() # Завершение программы

sys.version # Версия Python

sys.platform # Платформа: Windows, linux, darwin (macOS)

sys.path # Пути поиска модулей

sys.stdin.read() # Ввод из консоли

sys.stdout.write() # Вывод в консоль

sys.getdefaultencoding() # Текущая кодировка
```
