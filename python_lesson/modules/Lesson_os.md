# Lesson module `os` of Python

**В данном уроке мы изучим библиотку os**

## Начинаем изучение

- Что за модуль `os`? os - модуль стандартной библиотки Python, для работы с файлами и системой

- Подключаем так:

```python
import os
```

### Функции

- Давайте изучим функции `os`:

```python
import os

os.getcwd() # Текущая папка
os.listdir() # Список директорий
os.mkdir() # Создать папку
os.makedirs() # Создать папку со всем содержимым
os.remove() # Удалить пустой файл
os.unlink() # Удалить файл
os.rename() # Переименовать файл
os.path.exists() # Проверить существование
os.path.join() # Соединить пути
os.path.getsize() # Размер файла
os.system() # Выполнить команду в системе
os.chdir() # Сменить папку
os.rmdir() # Удалить пустую папку
os.removedirs() # Удалить всю папку (ОСТОРОЖНО!!!)
os.path.isfile() # Это файл?
os.path.isdir() # Это директория (папку)?
os.path.isabs() # Это абсолютный путь?
os.path.basename() # Имя файла
os.path.dirname() # Папка файла
os.path.split() # Разделить на папку + файл
os.path.splitext() # Разделить на файл + расширение
os.path.abspath() # Абсолютный путь
os.path.realpath() # Реальный путь
os.environ() # Переменные окружения
os.getenv() # Получить переменную
os.putenv(key, value) # Установить переменную
os.getpid() # Получить ID текущего процесса
```

- Отлично! Теперь вы знаете как работать с os, я вам также рекомендую изучить для комфортной работы с файлами - другие модули (shutil, pathlib: Path)
