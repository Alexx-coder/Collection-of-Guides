- Теперь основные команды в терминале программисту:


```bash
cd папка # Перейти в папку
cd .. # Перейти в прошлую папку (выше)
cd ~ / cd %USERPROFILE%  # Перейти домой
pwd # Текущая папка

ls / dir # Список файлов
mkdir # Создать папку (как в Python)
rmdir name # Удаление пустой папки
rm -rf name # Удалить всё содержимое папки
touch name # Создать файл
cat name # Прочитать файл
rm name # Удалить папку
cp src dst # Копировать
mv src dst # Переместить/Переименовать
nano name # Внутрений редактор

grep text name # Найти в файле
find . -name "*.py" # Найти файлы по фильтру (имени)
ls | grep py # Список + фильтр

whoami # Текущий пользователь
hostname # Имя компьютера
uname -a # Информация о ОС
top / htop # Процессы
ps aux # Список процессов
kill PID # Убить процесс по его PID
df -h # Свободное место на дисках
free -h # Свободная оперативная память
uptime # Время работы системы

ping google.com # Проверка соединения
curl ipconfig.me # Внешний ip
ip a / ifconfig # IP-Адреса
netstat -an # Открытые порты

cls / clear # Очистить экран
history # Список истории команд
exit # Выйти
```

- Фух, знаю что слишком много, но программисту на Python и другим - это понадобится

- Работают команды: на Bash, cmd, zsh (Знайте не все может работать на cmd, zsh)
