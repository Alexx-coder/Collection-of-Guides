# Lesson 3 of Python

**В данном уроке вы научитесь управлять потоком**

## Начинаем изучение

### Логические операторы

- Что такое `логические операторы`? 

- Логические операторы относятся к типу данных `bool`

- Есть несколько видов операторов. Приступаем!

#### Логические операторы `True/False/None`

- Что за логические операторы?

- True: обозначает правду

- False: ложь

- None: не определенно, пусто, нет значения

- Используются они в условиях (поговорим в этом уроке позже)

- Синтаксис: пишите их начиная с большой буквы или произойдёт исключение (ошибка)

#### Логические операторы `and/or/not`

- Что за логические операторы?

- and: обозначает и

- or: обозначает или

- not: обозначает нет или неъ

- Используются также в условиях

- Синтаксис: пишутся они с маленькой буквы или произойдет исключение (ошибка)

#### Условия  `if/elif/else`

- Что за условия?

- Они созданы для управления потоком

- if: обозначает если

- elif: если еще

- else: еще

- Пример:

```python
name = 'python'

if name == 'python':
    pass # Заглушка
elif not name:
    pass
else:
    pass
```

- Например, вы хотите узнать равно ли 100 + 100 = 200, напечатайте:

```python
task = "100 + 100 = 200"
if task:
    print("Правда")
else:
    print("Неправда")
```
