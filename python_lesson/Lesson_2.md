### Первая функция: print()

- Самая первая функция - это: `print()`

- За что она отвечает? Оно выводит надпись в терминал

- Попробуйте создать файл в редакторе кода `lesson.py` и напечатайте:

```python
print("Hello World!") # Вывод: Hello World!
```

- `Запомните`: в кавычки ("" или '') помещаем надпись, которую хотим вывести, а если хотим вывести значение переменной, то пишите без кавычек и пишите имя вашей перемнной

- Попробуйте в том же файле:

```python
name = "How are you?"
print(name) # Вывод: значение переменной - How are you?
```

- И последний пример с print, используя f-строку:

```python
name = 'How are you'
print(f"Hello! {name}") # Вывод: Hello! How are you 
```

### Вторая функция input()

- За что отвечает `input()`? Он дает вводить пользователю

- Попробуйте в том же файле:

```python
input('> ') # Вы можете что-то ввести
```

- Также input() можно присвоить как переменную (позже, узнаете для чего)

- Пример:

```python
name = input("Введите своё имя: ") # Запросит ввести ваше имя
print(name) # Вывод: значения переменной name
```

### Типы данных

- В пайтоне существуют типы данных, которые надо знать

- О них мы и поговорим

#### Тип данных `str`

- Что такое `str`? str - сокращенно от слова `string` (переводя на русский - строка)

- Данынй тип данных соотвествует буквам, символам и тд - короче текст

- Пример:

```python
name = "Python" # str
```

#### Тип данных `int`

- Что такое `int`? int - сокращенно от слова `integer` (переводя на русский - целое число)

- Данный тип данных соотвествует целым числам и цифрам

- Пример:

```python
age = 100 # int
```

#### Тип данных `float`

- Что такое `float`? float - сокращенно от слова `floating point` (переводя на русский - число с плавающей точкой)

- Данный тип данных соотвествует десятичной дроби 

- Пример:

```python
pi = 3.14 # float
```

#### Остальные типы данных

- О остальных типах данных вы узнаете позже, но расскажу что за типы данных и где нужны

- `Hex`: это не тип, а функция преобразования

- `bytes`: тип данных, используется в бинарных файлах, байтах и криптографии (узнаете позже)

- `bool`: условия (узнаете о них в следующем уроке)

- `list`: списки (узнаете позже)

- `dict`: словари (узнаете позже)

- `tuple`: кортеж (узнаете позже)

### Третья функция len()

- Что такое `len()`? len() - сокращенно от `lenght` (переводя на русский - количество)

- Создана чтобы узнавать количество символов

- Пример:

```python
age = 100
print(len(str(age))) # Вывод: 3
```

### Четвертая функция `type()`

- Что такое `type()`? 

- Создана для определения типа данных 

- Пример:

```python
age = 100
print(type(age)) # Вывод: <class> int
```

### Математика

#### Работа с математическими вычислениями

- Напишите:

```python
print(1+1) # Вывод: 2 | Сложение
print(1-1) # Вывод: 0 | Вычитание
print(1*1) # Вывод: 1 | Умножение
print(1/1) # Вывод: 1 | Деление
```

#### Пятая функция `round()`

- Что такое `round()`?

- Создана для округления на ближайшее число

- Пример:

```python
pi = 3.14
print(round(pi)) # Вывод: 3
```
- Еще можно указать до скольких знаков округлить:

```python
pi = 3.14
print(round(pi, 1)) # Вывод: 3.1
```

#### Шестая функция `abs()`

- Что такое `abs()`?

- Создана для вывода положительной формы числа

- Пример:

```python
pi = -3.14
print(abs(pi)) # Вывод: 3.14
```
