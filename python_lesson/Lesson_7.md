## Кортежи `(tuple)`

- что такое Кортеж?

- Кортеж - это коробка с вещами, которую нельзя открыть

- Нельзя менять

- Быстрее списка

- Безопаснее

- Запишите:

```python
# Создать
coords = (10, 20)

# Создать из списка
coords = tuple([10, 20])

# Распаковать
x, y = coords           # x=10, y=20

# Сравнить
print((1,2) == (1,2))   # True

# Конкатенация
nums = (1,2) + (3,4)    # (1,2,3,4)

# Длина
length = len((1,2,3))   # 3
```

- Идем дальше!

### Списки `(list)`

- Что такое Cписки?

- Списки - коробка с вещами, которые можно менять

- Можно добавлять, убирать, менять

- Порядок сохряняется

- Нумерация с 0

- Запишите:

```python
# Создать
items = [1, 2, 3]

# Добавить в конец
items.append(4)          # [1,2,3,4]

# Добавить в любое место
items.insert(1, 99)      # [1,99,2,3,4]

# Удалить по значению
items.remove(99)         # [1,2,3,4]

# Удалить по индексу
popped = items.pop(0)    # popped=1, items=[2,3,4]

# Найти индекс
idx = items.index(3)     # 1

# Посчитать вхождения
cnt = items.count(2)     # 1

# Перевернуть
items.reverse()          # [4,3,2]

# Отсортировать
items.sort()             # [2,3,4]

# Скопировать
new_items = items.copy()

# Очистить
items.clear()            # []
```

- Идем дальше!

### Словари `(dict)`

- Что такое Словари?

- Словари - это как шкафчик с ячейками, у каждого свой ключ

- Работает на базе: Ключ - Значение

- Быстрый поиск по ключу

- Идеален для данных с именами

- Похожий тип у файлов .json

- Запишите:

```python
# Создать
user = {"name": "alexx", "age": 12}

# Получить (с защитой)
name = user.get("name", "Unknown")    # alexx
city = user.get("city", "Moscow")     # Moscow

# Все ключи
keys = user.keys()        # dict_keys(['name', 'age'])

# Все значения
vals = user.values()      # dict_values(['alexx', 12])

# Пары
for k, v in user.items():
    print(f"{k}: {v}")

# Удалить по ключу
age = user.pop("age")     # age=12, user={'name':'alexx'}

# Последний удалённый элемент
last = user.popitem()     # ('name','alexx')

# Обновить/добавить несколько
user.update({"city": "New-York", "status": "admin"})

# Слить словари
a = {"x": 1}
b = {"y": 2}
a.update(b)               # {"x":1,"y":2}

# Очистить
user.clear()
```

- На этом всё!
