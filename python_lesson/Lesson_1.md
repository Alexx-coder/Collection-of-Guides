# Lesson 1 of Python

**Данный урок нужен для помощи в установке нужных инструментов (Python, Редактор кода), если у вас это есть можете пропустить**

## Установка инструментов

### Установка Python

- Для работы с python, вам нужен сам Python: [Ссылка на Python](https://www.python.org/downloads/)

- Выберите галочки: 

```text
Install for all users (если есть)
Add Python to PATH (ОБЯЗАТЕЛЬНО!)
Install launcher for all users
```

- Скачав, проверьте в вашем терминале (CMD/Powershell/Bash):

```bash
python --version # CMD/Powershell
python3 --version # macOS/Linux
```

- Готово: у вас есть Python!

### Установка Редактора кода

- Редактор кода, это обязательный инструмент для написания кода. Есть разные редакторы кода: PyCharm (JetBrains), Visual Studio Code (Microsoft) и тд.

- Чтобы скачать - вбейте в Браузер любой выше перечисленный редактор кода (Качайте от официального источника!!!)

- Если вы скачали Visual Studio Code: в расширениях (Extensions) - скачайте: Russian Launge for Visual Studio Code (Если вам нужен русский язык - данный пакет официальынй от Microsft) и Pylance, Python, Python Debugger, Python Environments (Вроде бы идут вместе - от Micrsoft пакет)

- Если скачали PyCharm: вам не придётся скачивать отдельные пакеты

- Надесь всё прошло удачно!

- `Удачного изучения и кода!!!`
