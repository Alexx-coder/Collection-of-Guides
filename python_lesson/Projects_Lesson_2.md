## Проекты

### Первая программа Auth

- Будет представлять себя программой для входа

- Создайте файл - `auth.py` и напечатайте:

```python
print("=== Auth ===")
name = input("[?] Имя: ")
print("[*] Теперь пароль")
pswd = input("[?] Пароль: ")
print("[+] Готово!")
print("=== Ваши данные ===")
print(f'[*] Имя: {name} | Пароль: {pswd}') # В реальных программах без вывода пароля - небезопасно
```

- Поздравляю! Первая программа...

### Вторая программа Type Class

- Для определения типа данных

- Создайте файл `type_class.py` и напечатайте:

```python
print("=== Type Class ===")
user_input = input("[?] Введите: ")
print("[*] Секунду")
print(f'[+] Результат: {type(user_input)}')
```

### Третья программа Round_Abs

- Для округления и получения положительного числа

- Создайте файл `round_abs.py` и напечатайте:

```python
user_input = input("[?] Введите число: ")

number = str(user_input)
print(f'[+] Результат: {round(number)} | {abs(number)}')
```
